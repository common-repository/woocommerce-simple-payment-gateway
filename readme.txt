A WooCommerce Extension that adds extra payment gateway - "Simple Payment Gateway"

== Description ==

This extension will add extra payment gateway - "Simple Payment Gateway" in your WooCommerce store.
Option for selecting shipping methods

= Installation =

1. Upload `Simple Payment Gateway` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
